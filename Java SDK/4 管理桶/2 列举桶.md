
以下代码用于列举所有的桶：

    ```language-java
     @Test
        public void testBucketList() {
            try {
                List<Bucket> buckets = s3.listBuckets();
                for (Bucket b : buckets) {
                    System.out.println("* " + b.getName());
                }
            }catch (AmazonS3Exception e) {
                System.err.println(e.getErrorMessage());
            }
        }
    ```
